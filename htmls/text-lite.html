Text Lite

Progressive Web Apps are steadily gaining ground in the online marketplace. As the web browser becomes an increasingly powerful tool for the average web user, with more features and capabilities being developed constantly. Progressive Web Apps are user experiences that have the reach of the web, load instantly and never show the down sour, even in uncertain network conditions respond quickly to user interactions with silky smooth animations and no janky scrolling, feel like a natural app on the device, with an immersive user experience. This new level of quality allows Progressive Web Apps to earn a place on the user's home screen. Text Lite is a progressive web application which allows the user to send text messages even when the internet is slow and also available everywhere. We all have faced some issues with a native application for chatting (for example, WhatsApp). Text Lite takes one login button to reach your chat screen irrespective of your Operating System and only needs a browser. The motive of this web application is to provide easy, fast and platform independent option for texting each other.As its title defines two words, '’text’’ for text messaging only and “lite” stands for lighter. It behaves like a native application, works for every single user regardless of browser choice.It allows the user to access their chat or text message offline, a user to "keep" it on their home screen without the hassle of an app store and has notification service as native applications.It provides security features for the user to ensure their privacy.
